# 仿小米在线商城
## 项目内容
模拟在线商城用户购物情景，实现用户的登陆登出，购物车的添加、删除以及最后形成订单的全过程。
## 项目说明
- 使用Vue.js框架搭建页面主体,使用Vue Router实现组件之间路由跳转和aixos获取资料。
- 使用express框架连接本地的MongoDB实现对商品数据以及用户购物资料的存储和删减。
- 使用Vuex管理多个组件的共同变量。
